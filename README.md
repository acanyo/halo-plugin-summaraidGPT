# 智阅全能AI助手

> 一款面向 Halo 的一站式 AI 内容与站点助手。它可以帮助站长完成摘要、写作、润色、标题和标签生成，也可以把文章、页面、文档和文件沉淀为知识库，让访客通过前台宠物助手直接向站点提问、检索资料并跳转到相关内容。

## 插件定位

智阅全能AI助手不是一个单纯的聊天窗口，而是围绕 Halo 内容站点构建的 AI 助手插件。

对站长来说，它是内容生产力工具，可以在写作、发布、摘要和 SEO 辅助上节省重复劳动。

对访客来说，它是站点里的智能向导，可以根据站内文章、页面、文档和知识库内容给出回答，并在需要时引导用户查看相关页面。

## 核心能力

### 内容生产

- 自动生成文章摘要，帮助读者快速了解内容重点。
- 根据主题生成文章草稿，适合辅助创作和灵感扩展。
- 对已有文本进行润色、改写和表达优化。
- 根据文章内容生成标题候选和推荐标签。

### 知识库问答

- 支持导入文章、页面、文档、Markdown、富文本和本地文件。
- 支持批量文件导入，标题默认使用文件名，减少手动填写成本。
- 支持 RAG 知识库检索，让 AI 基于站点资料回答问题。
- 支持关联资源展示，方便用户继续阅读原始内容。

### 前台宠物助手

- 支持前台宠物化入口，让 AI 能力自然出现在网站中。
- 支持小窗与全屏会话，适配轻量问答和深度阅读场景。
- 支持 Markdown 回答、复制、重试、停止生成和新会话。
- 支持配置助手名称、头像、欢迎语、快捷问题和主题配色。

### 站点 Agent

- 可以结合站内检索、页面内容和知识库一起回答问题。
- 可以根据用户意图引导访问相关文章、独立页面或评论区域。
- 可以在思考过程中提示当前正在检索、读取或整理的资料。
- 与 RAG 协同工作，避免把工具结果和知识库回答拆成多段重复输出。

### 管理与追踪

- 提供知识库导入、索引重建和任务状态查看。
- 支持会话记录与 AI 调用日志，方便排查和回溯。
- 支持导入任务、删除中状态和列表分页等管理体验。

## v4 版本亮点

v4 将插件从内容生成工具升级为完整的站点 AI 助手：

- 新增前台宠物助手，让访客可以直接和站点资料对话。
- 强化 RAG 知识库，支持更多内容来源和批量文件导入。
- 引入 Agent 站点能力，可以检索页面、定位资源并配合知识库输出。
- 优化小窗和全屏会话体验，支持插件配置的名称、头像和配色。
- 增加会话记录、AI 调用日志和导入任务状态，方便后台管理。

## 适合场景

- 内容站点希望为文章自动生成摘要、标题和标签。
- 博客或知识库站点希望让访客直接询问站内资料。
- 文档站点希望导入文件后通过 AI 进行检索问答。
- 个人博客希望通过宠物助手提供更有个性的前台交互。
- 站长希望统一管理 AI 调用记录、知识库导入和会话历史。

## 安装与启用

1. 在 Halo 后台进入「插件市场」，搜索「智阅全能AI助手」并安装。
2. 安装或启用 Halo AI Foundation，并配置可用的 AI 模型。
3. 进入插件设置，配置助手名称、头像、配色、模型和知识库参数。
4. 导入文章、页面、文档或本地文件，即可在前台助手中进行知识库问答。

## 环境要求

- Halo `>= 2.25.0`
- AI 能力需要安装并启用 Halo AI Foundation 插件
